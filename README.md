## Getting Started

Follow these steps to run the project locally:

### Prerequisites

- Node.js (>= 16.x)
- npm (>= 8.x) or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-org>/exchange-client-frontend.git
cd exchange-client-frontend

# Install dependencies
npm install
# or
yarn install
```
